# Design reference

`reference.html` is a self-contained export of the redesigned timeline produced in
Claude's design tool. It is **not** part of the build — it's kept purely as a **visual
and data reference** for porting the new look into the Vite + React app (`src/`).

It's a pre-bundled artifact (React + fonts + a copy of the data are inlined). To view
it, open the file in a browser or serve this folder statically. The live app continues
to render from `public/*.json`; this file does not.
